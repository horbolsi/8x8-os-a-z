# Security & Privacy
FlashTM8 ⚡️🌎🤖 | ©️8x8

- Secrets live **only** in an encrypted vault / environment variables — never hardcoded, never committed.
- All external communication is **anonymous**; sensitive media is DM-only.
- **Bounded autonomy**: trading, deploys and publishing require explicit gates.
- Every claim is backed by live verification; read-only audits never mutate state.
