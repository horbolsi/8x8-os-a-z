# Agents
FlashTM8 ⚡️🌎🤖 | ©️8x8

Agents are autonomous workers with **capability profiles**, **SOUL prompts**, and
access to the **shared brain**. They are not chatbots — they debate, hand off,
and self-improve.

## Core archetypes
- **HERMES** — Master orchestrator. Routes tasks, coordinates the fleet.
- **FLASH** — Creator proxy. Represents the sovereign owner.
- **AEGIS** — Security. Threat modelling and safe execution.
- **TRADER** — BitGet futures + multi-exchange screening.
- **ARCHITECT** — System design and capacity planning.
- **CODEx** — Builder. Code generation and review.
- **SCRIBE** — Writing and long-form content.
- **CANVAS** — Media, video and visual design.
- **SAGE** — Voice, audio and speech.
- **CURATOR** — Memory management and learning.

## How agents collaborate
- **Shared bridge**: `know / recall / send / score / task_queue`.
- **Debates**: Bull/Bear/Neutral consensus for trading and strategy.
- **Handoffs**: a chain of agents passes a task with context.
- **Learning**: successful patterns are saved as reusable skills.
