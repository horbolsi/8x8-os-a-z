# Overview — 8x8 OS v3.0
FlashTM8 ⚡️🌎🤖 | ©️8x8

8x8 OS is a sovereign AI operating system. It is not a single model or bot;
it is a **runtime for autonomous agency**: a layered stack where specialised
agents cooperate through a shared brain to perceive, decide and act.

## The one-line thesis
> Give agents bodies, a shared memory, and a mandate to improve — then let them run.

## What it does today
- Runs 131 agents across 8 archetypes (orchestrator, security, trading, writing,
  design, voice, intelligence, coordination).
- Connects to Telegram (6 bots), Discord, YouTube and the web.
- Trades on BitGet futures with risk-bounded automation.
- Produces multilingual content (text, image, voice, video) from a single studio.
- Self-documents, self-audits, and self-heals.

## Design pillars
1. **Sovereignty** — the creator owns and controls the whole stack.
2. **Autonomy** — runs 24/7 without owner intervention.
3. **Anonymity** — external presence never exposes the system or its secrets.
4. **Verification** — nothing is claimed without live proof.
5. **Continuity** — no session fragmentation, no lost state.
